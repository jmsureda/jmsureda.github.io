PYTEARCAT

PYthon TEnsor AlgebRa calCulATor

In the past couple of years I've been working in the development of PYTEARCAT. A python package designed to work with symbolic tensor calculus. The idea was born when, with Marco, we were developing perturbation equations in the context of a modified gravity theory and found ourselves with many issues using the existing software to calculate symbolic tensor operations at the time. For that reason, Marco started to develop what is now PYTEARCAT, and shortly after I joined. After a couple years, we released the first version of the package which is available for everyone.
